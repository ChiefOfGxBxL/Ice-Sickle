Ice Sickle Next-Generation WC3 Editor


Welcome to Ice Sickle! This application is pre v1.0, so it is still in major development.


Certain features work! You can open the Object Editor and edit various entities. A bare-bones Trigger Editor is also available, supporting only JASS for the time-being. Map compilation is partially implemented - it generates a few of the .w3x MPQ files which are then automatically added to a .w3x MPQ archive using Ladik's MPQEditor. Enjoy :)
